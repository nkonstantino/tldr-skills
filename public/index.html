Scheduler + Ping — Claude.ai skill integration via MCP

Step 1

Add to Claude.ai

Connect the MCP server to your Claude.ai project. Claude gets typed tool definitions — no manual HTTP construction needed.

https://tldr-skills.vercel.app/api/mcp
  1. 1Open your Claude.ai project → Settings → Integrations
  2. 2Click Add integration and paste the URL above
  3. 3Claude discovers the tools automatically — no further config needed
Step 2

What Claude can do

Once connected, Claude understands these tools natively and can invoke them from natural language.

list_tasks
Show all scheduled tasks
create_task
Schedule a new automated task
update_task
Pause, resume, or modify a task
delete_task
Remove a task permanently
run_task
Execute a task immediately
send_ping
POST a ping to any webhook
