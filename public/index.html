Step 1

Add to Claude.ai

Connect the MCP server to Claude.ai. Claude gets typed tool definitions — no manual HTTP construction needed.

https://tldr-scheduler.vercel.app/api/mcp
  1. 1Go to Settings → Connectors
  2. 2Click Add custom connector
  3. 3Enter a name (e.g. TLDR scheduler) and paste the URL above
  4. 4Click Add — Claude discovers the tools automatically
Step 2

What Claude can do

Once connected, Claude understands these tools natively and invokes them from natural language.

list_tasks
Show all scheduled tasks
create_task
Schedule a new automated task
update_task
Pause, resume, or modify a task
delete_task
Remove a task permanently
run_task
Execute a task immediately
send_ping
POST a ping to any webhook
